
## **TaskFlow – SaaS Project Management Platform (Full-Stack)**

Developed a production-grade **multi-tenant SaaS project management system** enabling organizations to collaborate efficiently through structured project and task workflows.

Designed and implemented a **role-based access control system (Owner/Member)** with secure authentication using **JWT-based stateless auth and HTTP-only refresh tokens**, ensuring scalable and secure session management.

Architected a complete **organization–project–task hierarchy**, where users join organizations via invite codes and collaborate within isolated workspaces.

Built dynamic **task assignment and workflow management system** with real-time status tracking across multiple stages (Kanban-style workflow).

Integrated **Redis-backed caching and queue system** to optimize performance and handle background operations with graceful fallback mechanisms.

Implemented secure **RESTful APIs using Node.js and Express**, with modular routing and middleware-based authentication and authorization.

Engineered seamless **frontend-backend integration** with environment-based API configuration, deployed frontend on **Vercel** and backend on **Render**, with MongoDB Atlas as the primary database.

Handled real-world production challenges including **CORS configuration, cross-domain authentication, environment variable management, and API routing consistency**.
